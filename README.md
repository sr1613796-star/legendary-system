# Space Shooter 🚀

A browser-based arcade space shooter game built with vanilla HTML5 Canvas and JavaScript. No dependencies, no build step — just open `index.html`.

## How to play

| Input | Action |
|-------|--------|
| Arrow Left / A | Move left |
| Arrow Right / D | Move right |
| Space | Shoot |
| Touch left third | Move left |
| Touch right third | Move right |
| Touch centre | Shoot |

## Features

- Waves of enemies that get faster and more numerous each level
- Elite enemies (orange) worth more points
- Enemy return fire that increases in speed per level
- Particle explosion effects
- Touch support for mobile

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your game will be live at `https://YOUR_USERNAME.github.io/REPO_NAME`
